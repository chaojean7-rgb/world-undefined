# 有用的链接

这里收集了我觉得有用的资源和链接。

## 如何添加链接？

在 `content/links/` 文件夹中创建新的 Markdown 文件，格式如下：

```markdown
---
title: "链接名称"
date: 2026-06-25
draft: false
description: "链接的简短描述"
link: "https://example.com"
category: "分类名称"
---

这里可以写更多关于这个链接的说明。
```
